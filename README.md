# Math Functions
Simple math functions calculator.

Included:
 * Factorial
 * Square Root
 * Cube Root
 * Fourth Root
 * Square
 * Cube
 * Power of 2
 * Circle Area
 * Sine
 * Cosine
 * Logarithm
 * Speed of light
 * Speed of sound in different media at different temperatures
